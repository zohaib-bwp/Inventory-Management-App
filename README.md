# Inventory-Management-App
📦 Product Management
   - Add new drinks (brand, size, purchase price, selling price)
   - Update stock when new supply arrives
   - Delete discontinued items
   - Track expiry dates for batches

👥 Customer Management
   - Store customer details (name, phone, address)
   - View purchase history
   - Track "pending payments" per customer

🧾 Invoice Generation
   - Create new invoices quickly
   - Select customer and products
   - Auto-calculate totals
   - Generate PDF invoices [citation:7]

📊 Stock Tracking
   - View total stock across all products
   - See remaining stock in real-time
   - Low stock alerts

💰 Reports
   - Daily sales summary
   - Pending payments report (your "Udhar Book")
   - Profit/Loss calculation [citation:8]
